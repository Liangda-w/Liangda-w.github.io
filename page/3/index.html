Leetcode 287： Find the Duplicate Number

Posted on 2020-06-25 Edited on 2020-07-22 In Leetcode , Medium Valine:
Symbols count in article: 5.2k Reading time ≈ 5 mins.

Description

Difficulty: Medium

Given an array nums containing n + 1 integers where each integer is between 1 and n (inclusive), prove that at least one duplicate number must exist. Assume that there is only one duplicate number, find the duplicate one.

Example:
Input: [1,3,4,2,2]
Output: 2

Note:

  1. You must not modify the array (assume the array is read only).
  2. You must use only constant, O(1) extra space.
  3. Your runtime complexity should be less than O(n^2).
  4. There is only one duplicate number in the array, but it could be repeated more than once.

Leetcode 130： Surrounded Regions

Posted on 2020-06-21 Edited on 2020-06-30 In Leetcode , Medium Valine:
Symbols count in article: 9.3k Reading time ≈ 8 mins.

Description

Difficulty: Medium

Given a 2D board containing 'X' and 'O' (the letter O), capture all regions surrounded by 'X'.
A region is captured by flipping all 'O's into 'X's in that surrounded region.

Example:

X X X X
X O O X
X X O X
X O X X

After running your function, the board should be:

X X X X
X X X X
X X X X
X O X X

Explanations:

Surrounded regions shouldn’t be on the border, which means that any 'O' on the border of the board are not flipped to 'X'. Any 'O' that is not on the border and it is not connected to an 'O' on the border will be flipped to 'X'. Two cells are connected if they are adjacent cells connected horizontally or vertically.
