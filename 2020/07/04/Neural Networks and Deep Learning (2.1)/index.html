Binary Classification 二分类

Logistic regression is an algorithm for binary classification problem, in which the result is a discrete value output.

Example: Cat vs Non-Cat

An image is store in the computer in three separate matrices corresponding to the Red, Green, and Blue color channels of the image. The three matrices have the same size as the image, for example, the resolution of the cat image is 64 pixels X 64 pixels, the three matrices (RGB) are 64 X 64 each.

The value in a cell represents the pixel intensity which will be used to create a feature vector of n dimension. In pattern recognition and machine learning, a feature vector represents an object, in this case, a cat or no cat. To create a feature vector, 𝑥, the pixel intensity values will be “unroll” or “reshape” for each color. The dimension of the input feature vector 𝑥 is:

(the total numbers we have in all of these matrix).

So in binary classification, the goal is to train a classifier that the input is an image represented by a feature vector 𝑥 and predicts whether the corresponding label 𝑦 is 1 or 0. In this case, whether this is a cat image (1) or a non-cat image (0).

Logistic Regression 逻辑回归

Logistic regression is a learning algorithm used in a supervised learning problem when the output 𝑦 are all either zero or one (for binary classification problems). The goal of logistic regression is to minimize the error between its predictions and training data.

Example Cat vs No - cat: Given an image represented by a feature vector 𝑥, the algorithm will evaluate the probability of a cat being in that image.

Given Inputs :

  • The input features vector: $x\in\mathbb{R}^{n_x}$, where 𝑛𝑥 is the number of features
  • The training label: $y\in\{0,1\}$

Parameters used in Logistic regression:

  • The weights: $w\in\mathbb{R}^{n_x}$, where 𝑛𝑥 is the number of features
  • The threshold: $b\in\mathbb{R}$

Output :

  • an output prediction $\widehat{y} = P(y=1\ |\ x)$ , where $0 \leq \widehat{y} \leq 1$
    • $\widehat{y} = w^Tx+b$ is a linear function (𝑎𝑥 + 𝑏) of input x for linear regression, but since we are looking for a probability constraint between [0,1], the sigmoid function is used

The function is bounded between [0,1] as shown in the graph above.
Some observations from the graph:

  • If 𝑧 is a large positive number, then 𝜎(𝑧) = 1

  • If 𝑧 is small or large negative number, then 𝜎(𝑧) = 0

  • If 𝑧 = 0, then 𝜎(𝑧) = 0.5

Logistic Regression Cost Function

To train the parameters 𝑤 and 𝑏 of the logistic regression model, we need to define a cost function.

Recap: $\widehat{y}^{(i)} = 𝜎(𝑤^T𝑥^{(i)} + 𝑏) = 𝜎(𝑧^{(i)})= \frac{1}{1+ 𝑒^{-z^{(i)}}}$

Given a training set of m training examples: $\{(x^{(1)},y^{(1)}), (x^{(2)},y^{(2)}) ,…,(x^{(m)},y^{(m)})\}$

Want: $\widehat{y}^{(i)} \approx y^{(i)}$

Loss (error) function

The loss function measures the discrepancy between the prediction $\widehat{y}^{(i)}$ and the desired output $y^{(i)}$. In other words, the loss function computes the error for a single training example (measure how good our $\widehat{y}$ is, when the true label is y), $L(\widehat{y},y)$ should be as small as possible:

  • Square error seems like a reasonable choice, except that it makes gradient descent not work well:
  • What we use:
  • If y = 1: $L(\widehat{y},y) = -log(\widehat{y}$), where we want $log(\widehat{y})$ large and $\widehat{y}$ large (should be close to 1);
  • If y = 0: $L(\widehat{y},y) = -log(1-\widehat{y})$, where we want $log(1-\widehat{y})$ large and $\widehat{y}$ small (should be close to 0);

Cost function:

The cost function is the average of the loss function of the entire training set (how well you’re doing on entire training set / the cost of the parameters).

We are going to find the parameters 𝑤 𝑎𝑛𝑑 𝑏 that minimize the overall cost function J.

Gradient(/Steepest) Descent Algorithm 阶梯(最速)下降法

Recap:

We are going to find the parameters 𝑤 𝑎𝑛𝑑 𝑏 that minimize the overall cost function J (w,b).

  • the height of the surface represents the value of J (w,b) at a certain point
  • J (w,b) is a convex function like a bowl

Steps of Gradient Descent:

  1. start at an initial point
  2. take iterative steps in the deepest downhill direction
  3. reach the global optimum

  • $\alpha$ : the learning rate, controls how big a step we take on each iteration or gradient descent
